# Duplicate Word Remover

A Python script that removes repeated words from a sentence.

## Description
This project reads a sentence, removes duplicate words, and preserves the original word order.

## Features
- Removes repeated words
- Preserves order
- Simple text processing logic

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python duplicate_words.py

4. Enter a sentence when prompted

## Example

Input:
python is easy and python is powerful

Output:
python is easy and powerful
